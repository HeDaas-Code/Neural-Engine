- 多元 `node if var [1:a,2:b,3:echo p_pick]`：
  - `If(cond=("var","var"), branches=[Branch(1, lookup(a)), Branch(2, lookup(b)), Branch(3, CallExpression("echo","p_pick"))])`
- 简略二元 `node [a?b:c]`：
  - `If(cond=("expr","a"), branches=[Branch(0, lookup(b)), Branch(1, lookup(c))])`

### 分支项解析规则

- `xxx`（裸变量）→ `NextDecl(var_name="xxx", target_id=lookup(xxx))`
- `node xxx` → 同上（去 node 前缀）
- `echo xxx` → `CallExpression(kind="echo", var="xxx")`
- `node echo xxx` → 同上
- `in ->xxx` / `in->xxx` → `CallExpression(kind="in", var="xxx")`
- 其他 → `ParserError("unknown branch '...' at line N")`
