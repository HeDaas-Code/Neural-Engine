- `isinstance(q, multiprocessing.Queue)` 判别（**不** duck type）
- `use_multiprocessing=True` + `None` → `multiprocessing.Queue()`
- `use_multiprocessing=False` + `None` → `queue.Queue()`
- 构造时**注入**两个 queue（测试用 mock）

### 序列化细节

- `multiprocessing.Queue` 接受 bytes / pickle — 用 **bytes**（`json.dumps(s).encode('utf-8')`）
- `queue.Queue` 接受任意 Python 对象 — 也走 **bytes** 路径统一
